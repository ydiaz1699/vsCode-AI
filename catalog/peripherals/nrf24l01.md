# NRF24L01+ — Transceptor 2.4GHz

## Resumen

| Parámetro | Valor |
|-----------|-------|
| Voltaje | 1.9 – 3.6V (**NUNCA 5V**) |
| Protocolo | SPI |
| Corriente | 12 mA TX / 14 mA RX |
| Level Shifter | No (pines 5V tolerant) |
| Frecuencia | 2.4 GHz |
| Alcance | ~100m (PA+LNA: ~1km) |
| Data Rate | 250kbps / 1Mbps / 2Mbps |

## Consideraciones Críticas

- 🚨 **NUNCA alimentar con 5V** — destruye el módulo
- 🚨 **Capacitor 10µF OBLIGATORIO** entre VCC y GND, lo más cerca posible del módulo
- Los pines de datos SÍ son tolerantes a 5V (lógica)
- Sin capacitor: comportamiento errático, no transmite, o se reinicia
- Versión PA+LNA alcanza ~1km en línea de vista
- Si no funciona: 90% de las veces es problema de alimentación

## Código de Ejemplo Mínimo

```cpp
#include <SPI.h>
#include <RF24.h>

#define CE_PIN  16
#define CSN_PIN 5

RF24 radio(CE_PIN, CSN_PIN);
const byte address[6] = "00001";

void setup() {
  Serial.begin(115200);
  if (!radio.begin()) {
    Serial.println("NRF24 no responde!");
    while (1);
  }
  radio.setPALevel(RF24_PA_LOW);
  radio.openWritingPipe(address);
  radio.stopListening();
}

void loop() {
  const char text[] = "Hola";
  bool ok = radio.write(&text, sizeof(text));
  Serial.println(ok ? "Enviado" : "Fallo");
  delay(1000);
}
```
