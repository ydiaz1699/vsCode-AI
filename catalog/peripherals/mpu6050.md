# MPU6050 — Acelerómetro + Giroscopio 6-DOF

## Resumen

| Parámetro | Valor |
|-----------|-------|
| Voltaje | 3.3 – 5V |
| Protocolo | I2C |
| Corriente | 3.9 mA |
| Level Shifter | No necesario |
| Dirección I2C | 0x68 (AD0=LOW), 0x69 (AD0=HIGH) |
| DOF | 6 (3 accel + 3 gyro) |

## Pines

| Pin | Función | Notas |
|-----|---------|-------|
| VCC | Alimentación | 3.3-5V (regulador onboard) |
| GND | Tierra | — |
| SDA | Datos I2C | — |
| SCL | Reloj I2C | — |
| AD0 | Dirección I2C | LOW=0x68, HIGH=0x69 |
| INT | Interrupción | Opcional, para DMP |

## Consideraciones Críticas

- ⚠️ **Requiere calibración inicial** — dejar inmóvil 2-3 segundos al arrancar
- DMP (Digital Motion Processor) integrado calcula ángulos sin MCU
- Para dos MPU6050: AD0 del segundo a HIGH → dirección 0x69
- Giroscopio tiene drift — complementar con filtro o fusión de sensores
- Módulos chinos suelen tener regulador y pull-ups integrados

## Código de Ejemplo Mínimo

```cpp
#include <Adafruit_MPU6050.h>
#include <Adafruit_Sensor.h>
#include <Wire.h>

Adafruit_MPU6050 mpu;

void setup() {
  Serial.begin(115200);
  if (!mpu.begin()) {
    Serial.println("MPU6050 no encontrado!");
    while (1);
  }
  mpu.setAccelerometerRange(MPU6050_RANGE_2_G);
  mpu.setGyroRange(MPU6050_RANGE_250_DEG);
  mpu.setFilterBandwidth(MPU6050_BAND_21_HZ);
  Serial.println("MPU6050 listo");
}

void loop() {
  sensors_event_t a, g, temp;
  mpu.getEvent(&a, &g, &temp);

  Serial.printf("Accel X:%.2f Y:%.2f Z:%.2f m/s²\n",
    a.acceleration.x, a.acceleration.y, a.acceleration.z);
  Serial.printf("Gyro  X:%.2f Y:%.2f Z:%.2f rad/s\n",
    g.gyro.x, g.gyro.y, g.gyro.z);
  Serial.printf("Temp: %.1f°C\n\n", temp.temperature);
  delay(500);
}
```
